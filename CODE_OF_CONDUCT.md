# Code of Conduct

This is a small, private project. The standard expected of everyone working on
it is simple: **be respectful, be constructive, act in good faith.**

## Expected behaviour

- Be courteous and professional in issues, pull requests, and reviews.
- Critique code and ideas, not people.
- Accept and give feedback gracefully; assume good intent.
- Respect that this is **proprietary, confidential** software (see
  [LICENSE](LICENSE)) — do not share code, credentials, protocol details, or
  hardware specifics outside the people authorised to work on it.

## Unacceptable behaviour

- Harassment, insults, or discriminatory language of any kind.
- Publishing others' private information.
- Disclosing or redistributing any part of this confidential codebase without
  written permission.

## Enforcement

Concerns can be raised privately with the maintainer via a GitHub private
vulnerability report or direct message. The maintainer may remove access, revert
contributions, or take any other action appropriate to keep the project healthy
and its contents confidential.

This is adapted in spirit from the [Contributor Covenant](https://www.contributor-covenant.org),
trimmed to fit a private, single-maintainer project.
